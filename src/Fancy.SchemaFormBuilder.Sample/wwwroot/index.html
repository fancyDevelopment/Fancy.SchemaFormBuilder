Schema Form Builder - Sample App

An app to demonstrate the use of the Fancy.SchemaFormBuilder project.

This app demonstrates how you can use the Fancy.SchemaFormBuilder project. You can find the sources and the documentation of the project here on GitHub. This sample app demonstrates how you could build a form and a rest controller to manage employees.
